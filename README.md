# Telegram Phone Info Bot

Простой Telegram-бот на Python. Он проверяет номер телефона через библиотеку
`phonenumbers` и показывает открытую справочную информацию: валидность,
регион, оператора и часовой пояс.

## Как создать бота

1. Открой Telegram и найди `@BotFather`.
2. Отправь команду `/newbot`.
3. Придумай имя и username бота.
4. Скопируй токен, который выдаст BotFather.

## Как запустить локально

```powershell
pip install -r requirements.txt
$env:BOT_TOKEN="токен_от_BotFather"
python bot.py
```

## Как залить на хостинг

1. Создай проект на хостинге.
2. Загрузи файлы проекта.
3. В настройках переменных окружения добавь:

```text
BOT_TOKEN=токен_от_BotFather
```

4. Укажи команду запуска:

```text
python bot.py
```

Токен нельзя публиковать в открытом доступе или отправлять посторонним.
